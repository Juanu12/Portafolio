Tecnologías

Python

Python es un lenguaje de programación interpretado, de alto nivel y de propósito general, creado por Guido van Rossum y lanzado por primera vez en 1991. Su diseño se centra en la legibilidad del código y la simplicidad en la sintaxis, lo que lo hace fácil de aprender y utilizar, incluso para principiantes.

  • Uso de diccionarios
  • Busqueda y creación de diccionarios
  • Creación y uso de funciones

Html y CSS

HTML es el lenguaje de marcado que define la estructura y el contenido de una página web, utilizando etiquetas para organizar textos, imágenes, enlaces, formularios y otros elementos., en cambio , CSS es el lenguaje que controla la apariencia y el diseño visual de una página web, aplicando colores, fuentes, márgenes, animaciones y adaptando el sitio a diferentes dispositivos.

  • Creación de paginas web
  • Formularios
  • Diseño de paginas web
  • Crear disposición movil para las paginas

JavaScript

JavaScript es un lenguaje de programación interpretado, dinámico y orientado a objetos, utilizado principalmente para dar interactividad y dinamismo a las páginas web. Permite manipular el contenido HTML y CSS en tiempo real, responder a eventos del usuario, realizar peticiones a servidores y crear aplicaciones web complejas. Es compatible con la mayoría de navegadores y se usa también en el desarrollo backend (con Node.js), aplicaciones móviles y de escritorio.

  • Llamar apis
  • Creacion de eventos
  • Eventlistener
  • Uso de elementos de html desde js

MongoDB

MongoDB es un sistema de gestión de bases de datos NoSQL, orientado a documentos, que almacena la información en formato BSON (similar a JSON). Se caracteriza por su alta escalabilidad, flexibilidad y rendimiento, permitiendo manejar grandes volúmenes de datos de manera eficiente.

  • Transacciones
  • Consultas
  • Funciones
  • Creacion y uso de colecciones

MySQL

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) que utiliza el lenguaje SQL (Structured Query Language) para almacenar, organizar y administrar datos. Se basa en un modelo de tablas y relaciones, lo que garantiza integridad y consistencia en la información.

  • Creacion y uso de tablas
  • Consultas
  • Modelos E-R
  • Estructuracion de datos
